# Dormant Tooling

This directory contains editor integrations, grammars, and supporting tooling for the DormantQL language.

## Structure

- `grammar/` — The portable grammar (Tree-sitter primary + TextMate secondary). Single source of truth.
- `vscode-dormantql/` — Official VS Code extension (syntax highlighting + language configuration).
- `docs/` — Editor integration guides.

See `docs/editor-integration.md` for usage instructions across different editors and platforms.
